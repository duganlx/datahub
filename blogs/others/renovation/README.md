频率 & 位宽
内存带宽 = 内存核心频率 x 内存总线位宽 x 倍增系数

双通道可以让位宽翻倍，两个内存好，插2，4槽，选D-Type布线，无残线导致的电气损失。

imc 内存控制器（CPU中的北桥?）

在其他因素不构成瓶颈的情况下，你购买的内存，其商品界面标注的频率为该内存的保底值，代表该内存厂商保证小于等于这个频率可以毫无压力运行，你可以自己手动尝试更高频率

在主板能不超频的情况下，内存频率的下限由内存本身决定，内存频率的上限由CPU默认内存频率与内存体质较低的那个决定

在对内存超频的情况下，内存频率将无视CPU的默认频率，此时内存频率的下限由内存本身决定，内存频率上限由内存体质、IMC性能/体质、主板布线与电气性能这三者最低的值决定

*风扇*

考虑维度：噪音、转速、散热

转速控制种类：DC（通过电压控制转速） 和 PWM（通过占空比来控制转速，开/关时间比）。调整的转速跨度 `PWM > DC`

分类：大4PIN定速扇，小3PIN风扇（测速、+12v、GND）DC调速，小4PIN风扇（多一根PWM调速线）。推荐买 小4PIN风扇

dB 噪声的标准计量单位: 分贝；dBA 是根据人类听觉属性设计的一种噪声测量单位。它采用 A 权重滤波器对噪声信号加权，以反映人类听觉系统对不同声压级频率的响应。在测量噪声时，通常使用dBA作为参照标准，以便更准确地衡量对人类听觉系统的影响。dB参考值: 安静的图书馆(30-40dB); 室内对话(50-60dB); 市区交通(70-80dB); 现场演唱会(90dB-100dB)

需要看清楚风扇的风向

机箱风扇的构建：底部进风，顶部出风；正面进风，尾部出风；

*屏幕*

OLED 有机自发光二极管
- 每个像素点可以单独控制开关，可以做到`息屏提醒`(部分点亮)
- 优点：对比度高，没有漏光，响应时间短，厚度薄，可弯曲，功耗低
- 缺点：寿命短；存在烧频；

LCD
- 一开屏幕整个背光层都需要打开
- 优点：寿命长
- 缺点：对比度低，存在漏光，响应时间长，厚度厚，不好弯曲，功耗高；理论分辨率达不到理论清晰度

对眼：LCD 蓝光 > OLED蓝光；OLED有PWM调光 存在频闪

PPI
- 公式 sqrt(x^2 + y^2) / 屏幕尺寸
- 300 够用；360 清晰；420 锐利
- 只适用RGB排列的LCD屏幕，不适用OLED
- OLED下 PPI = sqrt(x^2 + y^2) / 屏幕尺寸 x 81.65%，即 Pentitle排列的清晰度 = RGB排列的81.65%

OLED采用Pentitle原因：像素点种植成本、屏幕老化问题

micro-LED：将OLED的有机自发光二极管替换成无机材料的LED灯珠的屏幕

色域
- 100% sRGB ≈ 79.6% DCI-P3
- 100% sRGB 当下能提供最好的体验

色准：色彩实际颜色和显示器显示的颜色的差距
- ΔE 越小代表色彩偏离越小，色准越高
- ΔE ≤ 1.5     色准极好   S级   物理意义上几乎不存在色偏
- ΔE 1.5 - 3   非常优秀   A级   专业人士用肉眼也很难看出存在色偏
- ΔE 3 - 5     优秀       B级   日常使用几乎不会有影响
- ΔE 5 - 8     普通       C级   色彩敏感的人枯杨看出有偏色
- ΔE > 8       很差       D级   大部分人都能看出些微的偏色

更优先参看指标：分辨率、刷新率、响应时间、面板类型

色深
- FRC 通过类似PWM，色1和色2进行交替闪烁出现色1.5
- 6bit   平滑色彩过度会出现断层  不推荐
- 6抖8   细看会有静态噪点        凑合
- 8bit   够用不贵 性价比高       推荐
