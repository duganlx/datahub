备注
1. dB 噪声的标准计量单位: 分贝；dBA 是根据人类听觉属性设计的一种噪声测量单位。它采用 A 权重滤波器对噪声信号加权，以反映人类听觉系统对不同声压级频率的响应。在测量噪声时，通常使用dBA作为参照标准，以便更准确地衡量对人类听觉系统的影响。
2. dB参考值: 安静的图书馆(30-40dB); 室内对话(50-60dB); 市区交通(70-80dB); 现场演唱会(90dB-100dB)
